# Alipay SDK (支付宝SDK)


所有SDK源文件取自 [支付宝官网](https://docs.open.alipay.com/54/103419/)，原版未改动。

最后更新版本参见保留目录名

## 使用

```$bash
composer require miuhr/alipay-sdk
```
